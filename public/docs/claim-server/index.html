Claim Server

Cara membuat server bot WhatsApp melalui Polar.

Apa Itu Claim Server?

Claim Server adalah fitur Polar yang memungkinkan pengguna membuat instance bot WhatsApp menggunakan Polar Coin.

Setelah berhasil di-claim, server akan dibuat dan siap digunakan.

Persyaratan

  • Memiliki akun Polar
  • Memiliki Polar Coin cukup
  • Memiliki nomor WhatsApp aktif

Cara Claim Server

1. Login ke dashboard Polar
2. Buka menu Claim Server
3. Pilih script bot
4. Konfirmasi penggunaan Coin
5. Server dibuat otomatis

Setelah Server Aktif

Pengguna dapat menghubungkan WhatsApp, melihat status server, dan mengelola session melalui dashboard.
