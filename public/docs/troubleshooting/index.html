Troubleshooting

Solusi untuk masalah umum pada layanan Polar.

Bot Tidak Online

Jika bot tidak aktif, coba langkah berikut:

  • Periksa status server
  • Restart server melalui dashboard
  • Pastikan session WhatsApp masih aktif

WhatsApp Logout

Jika koneksi WhatsApp terputus:

1. Hapus session lama
2. Lakukan pairing ulang
3. Tunggu bot terhubung kembali

Pairing Code Tidak Muncul

  • Pastikan nomor benar
  • Gunakan format internasional (+62)
  • Coba beberapa saat lagi

Polar Coin Tidak Bertambah

Jika Coin belum masuk:

  • Pastikan proses shortlink selesai
  • Periksa koneksi internet
  • Cek kembali riwayat Coin

Masih Mengalami Masalah?

Hubungi customer support Polar untuk bantuan lebih lanjut.
