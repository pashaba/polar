Polar Dashboard

Tempat mengelola seluruh layanan bot WhatsApp Polar.

Apa Itu Dashboard?

Dashboard Polar adalah pusat kontrol pengguna untuk melihat informasi akun, mengelola server, dan memantau bot WhatsApp.

Menu Dashboard

  • Home — Ringkasan akun
  • Claim Server — Membuat server bot baru
  • Status — Melihat kondisi server
  • Sessions — Mengelola koneksi WhatsApp
  • History — Riwayat Polar Coin
  • Profile — Pengaturan akun

Status Server

Polar menyediakan monitoring server untuk melihat apakah bot sedang berjalan atau mengalami masalah.

🟢 Online = Server berjalan normal
🔴 Offline = Server perlu diperiksa

Mengelola Session

Session digunakan untuk menyimpan koneksi WhatsApp yang telah dipasangkan dengan bot.

  • Melihat session aktif
  • Menghapus session
  • Melakukan pairing ulang

Riwayat Polar Coin

Semua perubahan jumlah Coin akan dicatat, seperti mendapatkan Coin atau penggunaan Coin untuk claim server.
