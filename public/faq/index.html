Frequently Asked Questions

Pertanyaan yang sering ditanyakan mengenai Polar.web.id.

Apa itu Polar.web.id?

Polar.web.id adalah platform SaaS hosting bot WhatsApp yang memungkinkan pengguna menjalankan bot WhatsApp tanpa harus mengatur VPS sendiri.

Apakah Polar.web.id gratis?

Polar menyediakan layanan dengan sistem Polar Coin. Pengguna dapat mengumpulkan Coin untuk menggunakan layanan yang tersedia.

Apa itu Polar Coin?

Polar Coin adalah mata uang virtual internal yang digunakan untuk melakukan claim server dan mengakses fitur tertentu di Polar.

Bagaimana cara mendapatkan server bot?

Pengguna dapat login ke dashboard, memiliki Polar Coin yang cukup, kemudian melakukan claim server melalui menu yang tersedia.

Apakah membutuhkan VPS sendiri?

Tidak. Polar menyediakan server sehingga pengguna tidak perlu melakukan konfigurasi VPS secara manual.

Script apa saja yang tersedia?

Polar mendukung beberapa script bot WhatsApp seperti Phoenix MD, Ourin MD, dan Ourin MD Deluxe.

Bagaimana cara menghubungkan WhatsApp?

Pengguna dapat menggunakan metode pairing code untuk menghubungkan akun WhatsApp dengan bot.

Bagaimana jika bot mengalami masalah?

Silakan cek dokumentasi troubleshooting atau hubungi customer support Polar.
